# Custom GPT Template

This is a template for creating a new Custom GPT.

## Structure

- `instructions/` - GPT instructions and prompts
- `actions/` - OpenAPI schemas for custom actions
- `knowledge/` - Knowledge files (max 20 files per Custom GPT)

## Usage

Use the scaffold script to create a new GPT from this template:

```bash
npm run scaffold <gpt-name>
```
