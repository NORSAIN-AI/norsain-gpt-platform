# Domain Glossary & Terminology

## 1. Purpose
Provide authoritative definitions for all domain-specific terms, ensuring consistent interpretation by AI systems, engineers and analysts.

## 2. Terminology Table

| Term | Definition | Notes |
|------|------------|--------|
| Entity | A stable conceptual object in the domain | Must be versioned |
| Canonical Form | The standardized representation of an item | Defined in C-20 |
| Lifecycle | The sequence of domain phases | Defined in C-18 |
| Lineage | Chain of upstream sources | Logged in metadata |
| Artefact | Any output of a domain process | Versioned + auditable |

## 3. Abbreviations
| Abbrev | Meaning |
|--------|---------|
| CF | Canonical Form |
| QA | Quality Assurance |
| SOP | Standard Operating Procedure |

## 4. Synonyms & Disambiguation
- “Record” vs. “Item”:  
  → Item = domain entity  
  → Record = storage representation
