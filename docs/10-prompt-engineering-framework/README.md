Prompt Engineering Framework
===========================

This directory contains the Prompt Engineering Framework used by the project.

See `INDEX.md` for a table of contents and navigation between sections.

Structure highlights:
- `00-frontpage` – overview and quick links
- `01-prompt-engineering-core` – core guidance and patterns
- `02-examples` – practical prompt examples
- `03-evaluation` – evaluation methodology and metrics
- `04-version-control` – versioning and diffing guidelines
- `05-templates` – prompt templates for common tasks
- `06-schema-library` – schema contracts and LLM structure
- `07-troubleshooting` – common fixes and debugging tips
- `08-playbooks` – operational playbooks and runbooks

Contributions: open a PR against this branch and follow repository contribution guidelines.
