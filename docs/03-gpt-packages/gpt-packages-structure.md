# GPT Packages Structure (Draft)

## Mappestruktur

- `gpt-packages/instances/`
- `gpt-packages/.sandbox/`
- `gpt-packages/.archive/`
- `gpt-packages/templates/`

## Innhold i én GPT-pakke

- `gpt.json`
- `instructions/`
- `knowledge/`
- `actions/`

## Kobling til CLI

- `npm run scaffold`
- `npm run validate`
- `npm run generate-index`

Målet er å standardisere onboarding av nye GPT-pakker.
