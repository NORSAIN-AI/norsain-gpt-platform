# llm-platform: Repo Identity and Scope

## Hva er llm-platform

- Source of Truth (SoT) for NORSAIN LLM-plattform
- Inneholder GPT-pakker (`gpt-packages/`)
- Dokumenterer GPT-designstandarder (UI, knowledge, actions)
- Agentspesifikasjoner (f.eks. Platform-Dev-Architect)

## Hva er llm-platform ikke

- Ikke produktkode
- Ikke QMS-/TraceTimber-domeneimplementasjon

## Relasjon til andre repoer

- `NORSAIN-AI/platform`: produkt-/plattformrepo
- Mulig fremtidig NGAS-/standards-repo

## Praktisk bruk

- Utvikling og test av GPT-er (sandbox → instances → archive)
- Lagring av standarder og agentbeskrivelser
